## 文章做了什么
- **背景**：强化学习（RL）已成为大型语言模型（LLM）智能体后训练的核心范式，但其轨迹级奖励信号对于长程交互仅提供粗粒度的监督。在线策略自蒸馏（OPSD）通过引入带有特权上下文的教师分支提供的密集 token 级指导来补充 RL。
- **已有工作为什么解决不了**：将 OPSD 直接迁移到多轮智能体训练中存在两个主要问题：首先，多轮不稳定性会累积，导致监督信号 destabilize；其次，基于技能的特权指导需要不对称处理，因为教师的负面拒绝可能源于不完美的技能检索或利用，而非学生策略本身的错误。现有的混合方法（如 naive GRPO+OPSD）无法有效解决这些不稳定性及信任不对称问题。

## 文章的核心贡献点
- **提出了一个 SDAR (Self-Distilled Agentic Reinforcement Learning)**：一种将 OPSD 作为门控辅助目标，同时保持 RL 作为主要优化骨干的新型训练框架。
- **挑战1：多轮 OPSD 的不稳定性**。随着学生策略偏离教师支持的轨迹，token 级监督变得不可靠，导致 KL 散度激增和性能灾难性下降。SDAR 通过将分离的 token 级信号映射为 sigmoid 门控，仅在教师认可的正向差距 token 上加强蒸馏，从而稳定训练过程。
- **挑战2：特权指导中的不对称信任**。教师分支并非独立更强的模型，而是 augmented with privileged context 的同一策略，其负面拒绝可能由检索错误引起。SDAR 通过软衰减负向教师拒绝信号，避免了对因技能检索不完善导致的负面信号的过度惩罚，实现了对特权指导的不对称信任处理。

## 实现与部署，evaluation 结果，要有和相关工作对比
- **实验设置**：在 Qwen2.5 和 Qwen3 系列模型上进行评估，数据集包括 ALFWorld、WebShop 和 Search-QA。
- **对比基线**：主要对比了 GRPO、naive GRPO+OPSD 以及其他混合 RL-OPSD 基线。
- **结果**：
  - SDAR 显著优于 GRPO：在 ALFWorld 上提升 +9.4%，在 Search-QA 上提升 +7.0%，在 WebShop-Acc 上提升 +10.2%。
  - 稳定性：SDAR 避免了 naive GRPO+OPSD 出现的不稳定性问题。
  - 泛化性：在不同模型规模下，SDAR  consistently 优于其他混合 RL-OPSD 基线。

#### 总结
本文针对多轮 LLM 智能体训练中 RL 监督稀疏和 OPSD 不稳定的问题，提出了 SDAR 框架。通过引入门控机制，SDAR 有效地利用了 OPSD 的密集 token 级指导，同时通过不对称处理教师信号解决了多轮累积误差和特权上下文带来的噪声问题。实验证明，SDAR 在多个基准任务上均取得了显著的性能提升，且训练更加稳定，为 LLM 智能体的后训练提供了一种高效的新范式。
