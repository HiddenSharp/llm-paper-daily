## 1. What the Paper Does
- **Background**: As LLM-based agents increasingly rely on external tools, it is critical to evaluate their ability to sustain tool-grounded reasoning beyond familiar workflows and short-range interactions.
- **Limitations of Existing Work**: Current agent benchmarks (e.g., BFCL, Tau2-Bench, SWE-bench) focus on individual API calls or multi-step tasks in familiar domains with recurring solution templates. High performance on these benchmarks may reflect memorization of domain conventions or short-range reactive tool use rather than the ability to infer and execute novel procedures in unfamiliar environments.

## Core Contributions
- **Proposed AgentEscapeBench**: An escape-room-style benchmark designed to test whether agents can infer, execute, and revise novel tool-use procedures under explicit long-range dependency constraints.
- **Challenge 1: Long-range State Tracking and Dependency Management**. The difficulty lies in managing a directed acyclic graph (DAG) of tools and items, requiring agents to track hidden states revealed incrementally and propagate intermediate results over multiple steps. The paper addresses this by designing tasks with varying difficulty tiers (dependency depths), forcing agents to maintain correct contextual state to solve the final puzzle.
- **Challenge 2: Out-of-Domain Tool Reasoning**. The challenge is that agents face unfamiliar tools and narrative clues, preventing reliance on pre-trained common patterns. The paper constructs an environment with real external function calls, requiring agents to infer parameters from narrative clues and execute unseen tool combinations, thereby testing true generalization capabilities.

## Implementation and Evaluation
- **Implementation**: AgentEscapeBench includes 270 instances across five difficulty tiers, supporting fully automated evaluation. Tasks involve invoking real external functions, tracking hidden states, propagating intermediate results, and submitting deterministically verifiable answers.
- **Evaluation Results**: Experiments with 16 LLM agents and human participants show a sharp performance drop as dependency depth increases. Human success rates decline from 98.3% at difficulty-5 to 80.0% at difficulty-25, while the best model drops from 90.0% to 60.0%.
- **Comparison and Analysis**: Trajectory analysis attributes model failures mainly to breakdowns in long-range state tracking, clue adherence, and intermediate-result propagation. This contrasts with existing benchmarks, revealing that current agents struggle with deep contextual dependencies despite handling local tool use effectively.

## 4. Conclusion
AgentEscapeBench fills the gap in evaluating LLM agents' long-range, tool-grounded reasoning in unfamiliar environments. By introducing DAG-based escape room tasks, it exposes limitations in current agents' state tracking and complex dependency handling, serving as a diagnostic testbed for improving general-purpose reasoning, action, and adaptation.
