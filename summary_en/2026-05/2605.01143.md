## 1. What the Paper Did
- **Background**: Large Language Model (LLM)-powered agents demonstrate strong capabilities in autonomous task execution, tool use, and multi-step reasoning. However, their increasing autonomy introduces new attack surfaces where adversarial interactions can manipulate agent behavior through direct prompt injection, indirect content attacks, and multi-turn escalation strategies.
- **Limitations of Existing Work**: Current defense strategies focus on prompt-level filtering and rule-based guardrails. These are often insufficient when risk emerges gradually across interaction sequences, as they lack the ability to model context over multi-turn interactions.

## Core Contributions
- **Proposed a Low-Latency Fraud Detection Layer**: Designed to detect adversarial interaction patterns in LLM-powered agents.
- **Challenge 1: Single-point detection fails to capture gradual risks**. Traditional methods judging single prompts struggle with complex attacks spanning multiple turns. **Solution**: The approach models risk over interaction trajectories using structured runtime features derived from prompt characteristics, session dynamics, tool usage, and execution context, rather than isolating individual prompts.
- **Challenge 2: High latency of LLM-based detectors hinders real-time deployment**. Using large models for security checks introduces significant latency. **Solution**: The method employs lightweight models (specifically XGBoost) with 42 structured features, enabling low-latency real-time deployment while maintaining high detection accuracy.

## Implementation and Evaluation
- **Implementation & Data**: A synthetic corpus of 12,000 multi-turn agent interactions was constructed using parameterized templates to simulate realistic agentic workflows. The detection layer utilizes 42 structured features and an XGBoost classifier.
- **Evaluation Results**: The proposed detector achieves performance more than 9 times faster than LLM-based detectors.
- **Comparison**: Compared to traditional rule-based methods that focus on single-point filtering, this approach effectively identifies gradually evolving adversarial patterns through interaction-level behavioral detection, offering the low latency required for practical deployment.

## 4. Conclusion
This paper proposes a complementary defense mechanism: a low-latency fraud detection layer for LLM agents. By extracting structured features from interaction trajectories and using lightweight machine learning models, it addresses the inadequacy of traditional rule-based methods against multi-turn progressive attacks and the high latency of LLM-based detectors. Experiments suggest that interaction-level behavioral detection should become a core component of deployment-time defenses for LLM-powered agents.
