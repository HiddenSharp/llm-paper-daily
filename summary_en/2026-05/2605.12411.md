## 1. What the Paper Did
- **Background**: AI agents increasingly negotiate and transact in natural language with unfamiliar counterparts (e.g., a buyer bot facing an unknown seller). In these interactions, the counterpart's underlying LLM, prompts, control logic, and rule-based fallbacks are hidden, yet each decision carries monetary consequences.
- **Why Existing Work Falls Short**: Current methods typically rely on direct LLM prompting (LLM-as-Predictor) or simple game state and text features. However, direct prompting struggles to effectively capture opponent-specific behavioral patterns from limited interactions and fails to leverage decision-relevant signals embedded in hidden layers. Additionally, real-world market logs are scarce and confounded, making systematic evaluation difficult.

## Core Contributions
- **Proposed a**: Target-Adaptive Text-Tabular Modeling framework, which formulates opponent prediction as a tabular prediction task and incorporates an "LLM-as-Observer" feature extraction mechanism.
- **Challenge 1: How to quickly adapt to and predict the behavior of an unfamiliar opponent from very few interactions?**
  - **Difficulty**: The opponent's internal logic is a black box, and only a small number of prior game records (K samples) are available, making traditional fine-tuning costly and data-inefficient.
  - **Solution**: The paper constructs each decision point as a table row combining structured game state, offer history, and dialogue. It provides K previous games of the same target agent as labeled adaptation examples in the prompt, leveraging a tabular foundation model for few-shot adaptation.
- **Challenge 2: How to extract deeper decision intent signals beyond surface-level text?**
  - **Difficulty**: Direct text embeddings or explicit predictions often overlook the strategic reasoning processes implicit within the LLM, limiting prediction accuracy.
  - **Solution**: Introduces the "LLM-as-Observer" mechanism, where a small frozen LLM reads the decision-time state and dialogue. Its generated answer is discarded, and its hidden state is extracted as a decision-oriented feature. This turns the LLM into an encoder rather than a direct predictor, exposing decision-relevant signals that direct prompting misses.

## Implementation and Evaluation
- **Experimental Setup**: Studied in controlled bargaining and negotiation games. The training set includes 13 frontier LLM agents, and the test set includes 91 held-out scaffolded agents.
- **Baselines**: Compared against direct "LLM-as-Predictor" prompting and baselines using only "game state + text features."
- **Results**:
  - The full model outperforms both direct LLM prompting and traditional feature baselines across both tasks.
  - With K=16 (16 historical game samples), "Observer" features improved response prediction AUC by approximately 4 points.
  - Bargaining offer prediction error was reduced by 14%.
  - Results demonstrate that formulating opponent prediction as a target-adaptive text-tabular task enables effective adaptation, and hidden LLM representations expose signals not reliably surfaced by direct prompting.

## 4. Conclusion
This paper proposes a novel text-tabular modeling approach for predicting AI agent decisions under limited interaction. By combining structured game data with hidden representations extracted via "LLM-as-Observer," the method significantly improves prediction accuracy without accessing the opponent's internal logic. The study validates the effectiveness of using LLM hidden states as feature encoders for few-shot opponent modeling, offering a new technical pathway for prediction in black-box agent interactions.
