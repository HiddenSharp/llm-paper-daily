## 1. What the Paper Does
- **Background**: Accurate crop yield forecasting is critical for commercial soft fruit production (e.g., strawberries) but is constrained by the lack of sensor networks, satellite imagery, and high-resolution meteorological data in typical commercial farm records.
- **Why Existing Work Fails**: State-of-the-art models assume rich data inputs that are unavailable to most growers. Furthermore, existing numerical models (like XGBoost) lack explicit understanding of agricultural domain knowledge, leading to systematic errors that violate biological constraints (e.g., predicting yield before flowering). Acquiring new data is costly and impossible for historical records, while retraining models does not inherently fix these semantic logical errors.

## Core Contributions
- **Proposed a Structured LLM Agent Framework for Post-Hoc Correction**: This framework encodes agricultural domain knowledge into tools for phase detection, bias learning, and range validation to correct predictions from existing models without retraining or new data.
- **Challenge 1: Predictions violating seasonal and biological constraints**. It is difficult for traditional time-series models to enforce semantic rules like "no yield before flowering." The paper addresses this by using an agent tool for "phase detection" to identify the crop's growth stage and correct predictions that contradict known seasonal behaviors.
- **Challenge 2: Systematic biases and lack of dynamic calibration**. Static models fail to adapt to specific plot-level error patterns. The method solves this by employing "bias learning" to analyze recent error history and "range validation" to ensure predictions remain within plausible training distributions, dynamically adjusting the output.

## Implementation and Evaluation
- **Implementation**: The framework uses an LLM agent integrated with specific tools. Experiments showed that Llama 3.1 8B served as the most effective agent, while LLaVA 13B showed inconsistent gains, highlighting sensitivity to the refinement model choice.
- **Evaluation Results**: Evaluated on a proprietary strawberry dataset and a public USDA corn dataset. On the strawberry dataset, agent refinement reduced MAE by 20% and MASE by 56% for XGBoost baselines.
- **Comparison with Related Work**: The approach demonstrated consistent improvements across various baselines: Moirai2 (MAE -24%, MASE -22%) and Random Forest (MAE -28%, MASE -66%). This proves the framework's generalizability and superiority over using standalone SOTA models.

#### Summary
This paper addresses the gap between data-rich academic models and data-scarce commercial farming realities by proposing an LLM agent-based post-hoc correction framework. By integrating domain knowledge via phase detection, bias learning, and range validation, it effectively corrects systematic errors and biological constraint violations in standard forecasts. The method significantly improves accuracy across multiple models (XGBoost, Random Forest, Moirai2) without requiring additional data collection or model retraining, offering a practical solution for real-world agricultural deployment.
