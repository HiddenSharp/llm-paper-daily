## 1. What the Paper Does
- **Background**: In long-horizon agentic tasks, conversation history accumulates faster than fixed context windows can hold, making memory management critical for both answer accuracy and serving cost.
- **Limitations of Existing Work**: Current approaches either expand context windows without addressing retrieval relevance, perform heavy ingestion-time fact extraction at high token costs, or rely on heuristic graph traversal that fails to optimize both accuracy and efficiency, leaving the high-accuracy/low-cost regime underexplored.

## Core Contributions
- **Proposed PRISM Framework**: A training-free retrieval-side framework that treats long-horizon memory as a joint retrieval-and-compression problem over graph-structured memory.
- **Challenge 1: How to precisely locate relevant evidence under strict context budgets?**
  - **Difficulty**: Traditional graph traversal lacks query intent awareness, leading to noise inclusion or missed key information, wasting context or reducing accuracy.
  - **Solution**: PRISM introduces "Hierarchical Bundle Search" and "Query-Sensitive Edge Costing," aligning traversal with detected query intent via min-cost selection over typed relation paths to surface precise evidence.
- **Challenge 2: How to balance retrieval quality with computational/token costs?**
  - **Difficulty**: High-precision retrieval often relies on expensive LLM re-ranking, while low-cost methods sacrifice accuracy.
  - **Solution**: PRISM combines "Evidence Compression" and "Adaptive Intent Routing," compressing candidate bundles into compact contexts and routing most queries through zero-LLM tiers, achieving a Pareto-efficient balance between quality and efficiency.

## Implementation and Evaluation
- **Implementation**: PRISM is a plug-and-play retrieval framework requiring no fine-tuning or changes to upstream ingestion pipelines. It integrates four orthogonal inference-time components: hierarchical search, intent-aware costing, compression, and adaptive routing.
- **Evaluation Results**: On the LoCoMo benchmark, PRISM delivers substantially higher LLM-judge accuracy than all same-protocol baselines while using an order-of-magnitude smaller context budget.
- **Comparison**: Unlike MemoryBank, MemGPT, GraphRAG, and MAGMA, PRISM occupies a previously empty corner of the accuracy-context-cost frontier, demonstrating superior trade-offs between answer quality and retrieval efficiency.

## 4. Conclusion
PRISM successfully addresses memory management in long-horizon agents by formulating retrieval as min-cost selection over typed path templates paired with LLM-side compression. It achieves high-accuracy evidence retrieval at minimal context cost without training, offering an efficient and economical solution for deploying long-horizon agentic systems.
